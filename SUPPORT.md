## Getting Help

- **Bug reports and feature requests:** [GitHub Issues](https://github.com/InkyQuill/x-skills/issues)
- **CLI help:** run `x-skills --help` or `x-skills <command> --help` for built-in documentation
- **Diagnostics:** run `x-skills doctor` to check for common configuration and symlink issues
